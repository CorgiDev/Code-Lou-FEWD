#### Additional Software
- [Postman](https://www.postman.com/) - Used for testing API calls. More for when you get into advanced courses and start including an API.
- [Azure Data Studio](https://docs.microsoft.com/en-us/sql/azure-data-studio/what-is?view=sql-server-ver15) - Serves multiple purposes. Allow you to connect to SQL databases and create notebooks where you can store scripts just for notes or to actually run within the program.
- [LINQPad](https://www.linqpad.net/) - Similar to Azure Data Studio, but with a focus on LINQ and SQL. You can store scripts in it and run them against various databases.
